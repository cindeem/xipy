class xipy.slicing.image_slicers.ResampledVolumeSlicer(image, bbox=None, mask=False, grid_spacing=None, fliplr=False, **interp_kws)¶

Bases: xipy.slicing.image_slicers.VolumeSlicerInterface

This object cuts up an image along the axes defined by its CoordinateMap target space. The ResampledVolumeSlicer provides slices through an image such that the cut planes extend across the three {x,y,z} planes of the target space. Each plane is cut from a fully resampled image array.

SampledVolumeSlicer¶

class xipy.slicing.image_slicers.SampledVolumeSlicer(image, bbox=None, mask=False, grid_spacing=None, fliplr=False, interpolation_order=3)¶

Bases: xipy.slicing.image_slicers.VolumeSlicerInterface

This object cuts up an image along the axes defined by its CoordinateMap target space. The SampledVolumeSlicer provides slices through an image such that the cut planes extend across the three {x,y,z} planes of the target space. Each plane is sampled from the original image voxel array by spline interpolation.

cut_image(loc, axes=(0, 1, 2), oriented=True, **interp_kw)¶

Return len(axes) planes, which are cut along the axes specified. By default, the planes will be rotated if necessary to be aligned in the traditional orientation:

axial slice: (left-right by posterior-anterior) coronal slice: (left-right by inferior-superior) sagittal slice: (posterior-anterior by inferior-superior)

To return planes as-they-are-sliced in the array, set oriented=False

Parameters:

loc : iterable, len-3

The coordinates of the cut location

axes : iterable, len-1, 2, or 3

The returned planes will be those normal to these axes (by default all three SAG, COR, AXI axes – also may be specified by name in terms of ‘SAG’, ‘COR’, ‘AXI’, or ‘x’, ‘y’, ‘z’)

aligned : bool

Whether to return the planes aligned to the canonical orientations

interp_kw : dict

Keyword args for the interpolating machinery (ie, ndimage.map_coordinates keyword args–not implemented in all subclasses)
