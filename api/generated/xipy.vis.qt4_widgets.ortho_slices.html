MplQT4OrthoSlicesWidget¶

class xipy.vis.qt4_widgets.ortho_slices.MplQT4OrthoSlicesWidget(parent=None, **kwargs)¶

Bases: xipy.vis.qt4_widgets.auxiliary_window.TopLevelAuxiliaryWindow

This class is a Qt4 panel displaying three SliceFigures which cut into the standard orthogonal planes of a volumetric medical image. The voxel of intersection is manipulated by moving crosshairs belonging to each SliceFigure object, and whose positions are linked up by this class.

This class can load/refresh base images and overlay images into the three SliceFigures, and emits PyQt4 signals when the intersecting voxel changes.
